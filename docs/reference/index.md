# Reference

The complete technical reference for IntentText v2.11.

## Keywords

IntentText has **55 keywords** across 7 categories. Each keyword defines a block type.

- [All Keywords](./keywords) — sortable, filterable table of all 55 keywords
- [Document Identity](./keywords/document-identity) — `title:`, `summary:`, `meta:`, `context:`, `track:`
- [Content](./keywords/content) — `note:`, `quote:`, `warning:`, `tip:`, `code:`, `image:`, `link:`, `cite:`, `def:`, `figure:`, `contact:`
- [Structure](./keywords/structure) — `section:`, `sub:`, `break:`, `group:`, `ref:`, `deadline:`
- [Data](./keywords/data) — `input:`, `output:`, `table:`, `metric:`
- [Agent](./keywords/agent) — `step:`, `gate:`, `trigger:`, `emit:`, `decision:`, `context:`, `memory:`, `prompt:`, `tool:`, `audit:`, `done:`, `error:`
- [Trust](./keywords/trust) — `approve:`, `sign:`, `freeze:`, `revision:`, `policy:`, `amendment:`
- [Layout](./keywords/layout) — `page:`, `font:`, `header:`, `footer:`, `watermark:`, `signline:`
- [Aliases](./keywords/aliases) — all 47 aliases mapped to canonical keywords
